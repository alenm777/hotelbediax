# HotelBediaX - Destinations Module

Aplicación web para gestionar destinos turísticos (CRUD, filtros, paginación).

## Tecnologías
- Frontend: React (SPA)
- Backend: Node.js + Express
- Base de datos simulada: JSON
- Routing: React Router
- Estilos: CSS en línea y Flexbox

## Instalación

1. Clonar el repositorio:

## Backend:
cd backend
npm install
npm run dev

## Frontend:
cd ../frontend
npm install
npm run dev


## Funcionalidades:

-Crear, editar y eliminar destinos
-Filtrar por nombre, país y ciudad
-Paginación
-Tabla con estrellas para mostrar rating
